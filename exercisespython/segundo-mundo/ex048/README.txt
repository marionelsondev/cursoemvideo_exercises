Faça um programa que calcule a soma entre todos os números que são múltiplos de três e que se encontram no intervalo de 1 até 500.

Raciocínio:

1. Desenvolver um loop for para iterar sobre os números entre 1 até 500. -
2. Desenvolver uma condicional para verificar se o número é múltiplo de três. -
3. Desenvolver o cálculo da soma caso a condição seja atendida. -
4. Printar na tela o resultado para terminar o loop. -
