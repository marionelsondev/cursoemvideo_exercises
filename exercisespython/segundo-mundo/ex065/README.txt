Crie um programa que leia vários números inteiros pelo teclado. No final da execução, mostre a média entre todos os valores e qual foi o maior e o menor valores lidos. O programa deve perguntar ao usuário se ele quer ou não continuar a digitar valores.

Raciocínio:

1. Ler um número inteiro. -
2. Calcular a média dos valores inseridos -
3. Registrar quantos números foram digitados no total -
4. Registrar qual foi o maior valor inserido -
5. Registrar qual foi o menor valor inserido -
6. Perguntar ao usuário se ele que digitar mais valores. (Utilizar o loop while) -
7. Exibir o resultado ao usuário {
    data: 
    "Você digitou 4 números e a média foi 5.75\n
    O maior valor foi 9 e o menor foi 2"
} -
