Melhore o jogo do DESAFIO 28 onde o computador vai "pensar" em um número entre 0 e 10. Só que agora o jogador vai tentar advinhar até acertar, mostrando no final quantos palpites foram necessários para vencer.

Raciocínio:

1. Computador irá escolher aleatoriamente um número entre 0 e 10. -
2. Utilize o loop while para permitir que o usuário digite um número inteiro até acertar o número escolhido pela máquina. -
3. No final exiba quantos palpites foram necessário para vencer. -
