Faça um programa que jogue par ou ímpar com o computador. O jogo só será interrompido quando o jogador perder, mostrando o total de vitórias consecutivas que ele conquistou no final do jogo.

Raciocínio:

1. Ler um número inteiro -
2. Ler o dado par ou ímpar do usuário -
3. O computador deve selecionar um número aleatório -
4. O programa finaliza quando o usuário perde (condição de parada) -
5. Exibir ao usuário quantas vitórias consecutivas ele conquistou -
