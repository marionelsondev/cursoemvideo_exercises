Crie um programa que leia dois valores e mostre um menu na tela.

Raciocínio:

1. Ler dois valores -
2. Criar um menu de opções com as seguintes opções: -
    [ 1 ] somar -
    [ 2 ] multiplicar -
    [ 3 ] maior -
    [ 4 ] novos números -
    [ 5 ] sair do programa -
3. Criar condicionais para executar as devidas operações quando escolhidas. -
4. Exibir o resultado da opção escolhida para o usuário. -
5. Utilizar um loop while para receber as escolhas do usuário até ele escolher parar. -
