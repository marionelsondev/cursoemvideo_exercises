Faça um programa que leia o peso de cinco pessoas. No final, mostre qual foi o maior e o menor peso lidos.

Raciocínio:

1. Criar um loop for para ler o peso 5 vezes. -
2. Criar condicional de comparação maior. -
3. Criar condicional de comparação menor. -
4. Exiba para o usuário qual foi o menor e o maior peso. -
