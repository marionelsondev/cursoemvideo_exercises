Escreva um programa que leia dois números inteiro e compare-os mostrando na tela uma mensagem: 

Qual é o maior valor, ou se for o caso dizer que são iguais.

Raciocínio:

1. Ler os dois valores.
2. Criar condicionais compostas para atender caso um número seja maior que o outro ou se são iguais.
3. Exibir o resultado para o usuário.
