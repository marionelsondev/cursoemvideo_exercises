Escreva um programa em python que leia um número inteiro qualquer e peça para o usuário escolher qual será a base de conversão: 1 para binário, 2 para octal e 3 para hexadecimal. 

Raciocínio:

1. Ler um número inteiro. -
2. Criar condicional de conversão para binário.
3. Criar condicional para conversão octal.
4. Criar condicional para conversão hexadecimal.
5. Exibir o resultado da conversão para o usuário.
