Desenvolva um programa que leia o nome, idade e sexo de 4 pessoas. No final do programa, mostre: a média de idade do grupo, qual é o nome do homem mais velho e quantas mulheres têm menos de 20 anos.

Raciocínio:

1. Utilizar o loop for para ler o nome, idade e sexo de 4 pessoas. -
2. Criar uma condicional para verificar se aquela pessoa é do sexo Masculino e se aquela é a maior idade masculina digitada naquele turno, caso seja será armazenado em uma variável o nome da pessoa. -
3. Criar uma condicional para caso a pessoa seja uma mulher e tenha menos de 20 anos, adicione +1 no contador de mulheres 20 -
4. Calcular a média de idade do grupo: Somar a idade em uma variável sempre que o loop finalizar e no final dividir por 4. -
5. Exibir o resultado ao usuário. -
