Faça um programa que leia 5 valores numéricos e guarde-os em uma lista. No final, mostre qual foi o maior e o menor valor digitado e as suas respectivas posições na lista.

Raciocínio:

1. Utilizar um loop for para ler 5 números inteiros -
2. Popule uma lista com os 5 valores lidos -
3. Use condições para distinguir qual foi o maior e o menor número inputado na lista -
4. Utilize o método index para descobrir a posição dos respectivos números na lista -
