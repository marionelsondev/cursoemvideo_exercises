Crie um programa onde o usuário possa digitar vários valores numéricos e cadastre-os em uma lista. Caso o número já exista lá dentro, ele não será adicionado. No final, serão exibidos todos os valores únicos digitados, em ordem crescente.

Raciocínio:

1. Utilizar o loop while para ler vários números inteiros -
2. Popular a lista com o primeiro número usando o método "append" -
3. Para popular a lista com os próximos números, utilize uma condicional para verificar se aquele número digitado já existe na lista. Se existir ele não deve ser adicionado novamente -
4. Mostre todos os valores únicos digitados usando o método "sort" para ordenar a lista em ordem crescente -
