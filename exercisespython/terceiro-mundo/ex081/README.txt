Crie um programa que vai ler vários números e colocar em uma lista.

A) Quantos números foram digitados
B) A lista de valores, ordenada de forma decrescente
C) Se o valor 5 foi digitado e está ou não na lista

Raciocínio:

1. Utilize um loop while para receber vários números inteiros.
2. Utilizar o método len para saber quantos valores foram digitados e exibir ao usuário
3. Utilizar o método sort com o parâmetro (reverse=True) para exibir ao usuário a lista ordenada em ordem decrescente
4. Utilizar uma condicional para verificar se o valor 5 foi digitado e exibir ao usuário se ele está ou não na lista.
