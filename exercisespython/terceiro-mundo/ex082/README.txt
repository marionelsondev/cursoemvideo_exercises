Crie um programa que vai ler vários números e colocar em uma lista. Depois disso, crie duas listas extras que vão conter apenas os valores pares e os valores ímpares digitados, respectivamente. Ao final, mostre o conteúdo das três listas geradas.

Raciocínio:

1. Utilizar um loop while para ler vários números inteiros.
2. Exibir a lista completa ao usuário.
3. Utilizar o loop for para remover os números ímpares e exibir a lista com os número pares ao usuário.
4. Utilizar o loop for para remover os números pares e exibir a lista com os números ímpares ao usuário.
